## Mind Gym 3D Assets

This folder contains baseline `.glb` scene assets used by `MindGym3DScene`.

Current files were downloaded from Khronos sample assets (free/open examples) to keep the app fully runnable without manual setup.

- `classroom.glb` -> exam scenarios
- `presentation.glb` -> presentation scenarios
- `conflict.glb` -> conflict scenarios
- `social.glb` -> social scenarios
- `stress.glb` -> academic stress scenarios
- `coach.glb` -> optional animated character

To replace with production-quality models, keep the same filenames and copy your own `.glb` files here.
