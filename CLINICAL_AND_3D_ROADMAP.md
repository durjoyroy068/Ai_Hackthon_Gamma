# ক্লিনিক্যাল নির্ভুলতা ও 3D — ধাপে ধাপে রোডম্যাপ

এই গাইডে **আপনার কাজ** এবং **ইতিমধ্যে কোডে করা কাজ** আলাদা করা আছে।

---

## ফেজ ১ — ক্লিনিক্যাল ভিত্তি (২–৪ সপ্তাহ)

### ধাপ ১: মনোবিজ্ঞান বিশেষজ্ঞ খুঁজুন
**আপনার কাজ:**
1. বিশ্ববিদ্যালয়ের Psychology/Counseling বিভাগে একজন ফ্যাকাল্টি বা কাউন্সেলরের সাথে যোগাযোগ করুন
2. তাকে দিন:
   - `Dataset/MindGym/SCENARIOS_FOR_EXPERT_REVIEW.md` (আমি জেনারেট করেছি)
   - `Dataset/MindGym/EXPERT_REVIEW_CHECKLIST.md`
   - `Dataset/MindGym/CRISIS_PROTOCOL_DRAFT.md`
3. তার সাইন-অফ/কমেন্ট পেলে CSV আপডেট করুন

**আমি করেছি:** রিভিউ ডকুমেন্ট + চেকলিস্ট + ক্রাইসিস প্রোটোকল ড্রাফট

---

### ধাপ ২: সিনারিও ও স্কোর রুব্রিক যাচাই
**আপনার কাজ:**
1. বিশেষজ্ঞ `02_scenario_templates.csv` পড়বেন
2. প্রতিটি সিনারিওতে চিহ্নিত করবেন: ✅ OK / ⚠️ সংশোধন / ❌ বাদ
3. `03_branching_choices.csv`-এ `score_impact` যাচাই (coping, avoidance, clarity)
4. সংশোধিত CSV প্রজেক্ট ফোল্ডারে সেভ করুন

**আপনার পরে চালান:**
```powershell
C:\php-8.3.12\php.exe backend\artisan db:seed --class=MindGymSeeder --force
```

**আমি করেছি:** `scripts/validate_mindgym_csv.py` — CSV আপলোডের আগে ভ্যালিডেশন

```powershell
python scripts/validate_mindgym_csv.py
```

---

### ধাপ ৩: বাংলা কন্টেন্ট রিভিউ
**আপনার কাজ:**
1. একজন মাতৃভাষী বাংলা স্পিকার (ইдеально psychology ব্যাকগ্রাউন্ড) `title_bn`, `opening_scene_bn`, `choice_text_bn` পড়বেন
2. অপ্রাকৃতিক বা ভুল টোন ঠিক করবেন
3. বিশেষত: SCN_004-এ typo আছে (`অarter` → `অর্ধ`) — ঠিক করুন

**আমি পারি না:** সাংস্কৃতিক/ভাষাগত রিভিউ — মানুষ লাগবে

---

### ধাপ ৪: ক্রাইসিস প্রোটোকল ফাইনালাইজ
**আপনার কাজ:**
1. `CRISIS_PROTOCOL_DRAFT.md` বিশেষজ্ঞের সাথে চূড়ান্ত করুন
2. কোন কীওয়ার্ডে হটলাইন দেখাবে তা নিশ্চিত করুন
3. আমাকে বলুন — আমি `safety_keywords.json` আপডেট করব

**আমি করেছি:** Mind Gym রিফ্লেকশনে সেফটি ডিটেকশন + জরুরি হেল্পলাইন UI

---

### ধাপ ৫: Gemini API Key
**আপনার কাজ:**
1. [Google AI Studio](https://aistudio.google.com/apikey) থেকে ফ্রি API key নিন
2. `backend/.env` ফাইলে যোগ করুন:
   ```
   GEMINI_API_KEY=your_key_here
   ```
3. সার্ভার রিস্টার্ট করুন

**আমি করেছি:** Gemini ইন্টিগ্রেশন — key দিলেই কাজ করবে

---

## ফেজ ২ — আসল ডেটা সংগ্রহ (৪–৮ সপ্তাহ)

### ধাপ ৬: বেটা টেস্ট শুরু
**আপনার কাজ:**
1. `Dataset/MindGym/BETA_FEEDBACK_FORM.md` দেখে Google Form বানান
2. ৫০–১০০ শিক্ষার্থীকে লিংক পাঠান (ক্লাস, ফেসবুক গ্রুপ, ক্যাম্পাস)
3. প্রতিটি টেস্টারকে Mind Gym-এ কমপক্ষে ৩টি সিনারিও খেলতে বলুন
4. ফর্ম রেসপন্স CSV ডাউনলোড করে আমাকে দিন

**আমি করেছি:** ফর্মের প্রশ্ন তালিকা (কপি-পেস্ট রেডি)

---

### ধাপ ৭: আসল সেশন লগ সংগ্রহ
**আপনার কাজ:**
1. অ্যাপ চালান — প্রতিটি সেশন DB-তে `mind_gym_sessions` টেবিলে সেভ হয়
2. বেটা শেষে এক্সপোর্ট করুন (অথবা আমাকে বলুন, আমি এক্সপোর্ট কমান্ড লিখে দেব)
3. `Dataset/MindGym/templates/04_real_session_logs_TEMPLATE.csv` ফরম্যাটে ডেটা রাখুন
4. `04_response_session_scores.csv` বদলে দিন (সিন্থেটিক ডেটা সরান)

**আমি করেছি:** টেমপ্লেট CSV + সেশন লগিং API

---

### ধাপ ৮: বিশেষজ্ঞ-লেবেলড উদাহরণ
**আপনার কাজ:**
1. বিশেষজ্ঞ ৫০–১০০টি রিয়েল রিফ্লেকশন লেবেল করবেন (good/bad + কেন)
2. `Dataset/MindGym/templates/05_expert_examples_TEMPLATE.csv` ফরম্যাটে পূরণ করুন
3. `05_good_bad_response_examples.csv`-এ যোগ করুন

**আমি করেছি:** টেমপ্লেট + ভ্যালিডেশন স্ক্রিপ্ট

---

### ধাপ ৯: AI মডেল রি-ট্রেইন (ঐচ্ছিক)
**আপনার কাজ:**
1. আপডেটেড `Dataset/` ফোল্ডার zip করুন
2. `Dataset/Mon_Songlap_Full_AI_Training_Colab.ipynb` Google Colab-এ খুলুন
3. Dataset আপলোড করে রান করুন
4. `mon_songlap_trained_outputs/` ডাউনলোড করে প্রজেক্টে কপি করুন

**আমি করেছি:** Colab নোটবুক + `scripts/generate_ai_config.py`

```powershell
python scripts/generate_ai_config.py
```

## ফেজ ৩ — 3D Mind Gym (৮–১৬ সপ্তাহ)

### ধাপ ১০: 3D স্কোপ ঠিক করুন
**আপনার কাজ — সিদ্ধান্ত নিন:**
| অপশন | খরচ | সময় | মান |
|-------|------|------|-----|
| A) Ready-made asset pack (Sketchfab/Unity) | কম | ২–৪ সপ্তাহ | মাঝারি |
| B) ফ্রিল্যান্স 3D আর্টিস্ট | মাঝারি | ৬–১০ সপ্তাহ | ভাল |
| C) Unity/Unreal full game | বেশি | ১২+ সপ্তাহ | সেরা |

**সুপারিশ:** শুরুতে **১টি সিনারিও** (SCN_001 পরীক্ষার হল) 3D-তে বানান, বাকি পরে

---

### ধাপ ১১: 3D Assets সংগ্রহ
**আপনার কাজ:**
1. প্রয়োজনীয় `.glb` ফাইল:
   - `exam_hall_room.glb` — পরীক্ষার হল
   - `student_character.glb` — স্টুডেন্ট (neutral pose)
   - `desk_chair.glb` — ডেস্ক/চেয়ার
2. `public/models/mind-gym/` ফোল্ডারে রাখুন
3. আমাকে জানান — আমি 3D viewer ইন্টিগ্রেট করব

**ফ্রি রিসোর্স:**
- [Sketchfab](https://sketchfab.com) — CC-licensed models
- [Kenney.nl](https://kenney.nl) — game assets
- [Poly Pizza](https://poly.pizza) — low-poly GLB

---

### ধাপ ১২: 3D ইন্টিগ্রেশন (আমি করব — assets পেলে)
আপনি assets দিলে আমি করব:
- Three.js / React Three Fiber viewer
- সিনারিও অনুযায়ী scene load
- Choice UI overlay (বর্তমান Lite ফ্লো রাখা)

**আপনার কাজ:** শুধু `.glb` ফাইল + কোন সিনে কোন মডেল লাগবে তার তালিকা
